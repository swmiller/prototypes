# Prototypes Repository

This repository contains a collection of small, focused prototype applications and experiments. These prototypes explore ideas across multiple domains — such as media ingestion, FHIR tooling, AI moderation pipelines, Azure messaging patterns, and architectural patterns.

Most of these prototypes are isolated, self‑contained experiments. Some are simple console‑app thought exercises; others may be more complete vertical slices that eventually evolve into production‑ready components elsewhere.
